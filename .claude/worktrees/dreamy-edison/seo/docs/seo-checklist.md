# On-Page SEO Checklist

Last updated: 2026-04-13

Use this checklist before marking any article as SEO-optimized.

## Content

- [ ] Primary keyword appears in first 100 words
- [ ] Primary keyword in H1
- [ ] Secondary keywords in H2/H3 subheadings
- [ ] SERP-recommended terms woven in naturally (via `/neuronwriter-seo` or `/beuron-seo`)
- [ ] Content matches search intent (informational, transactional, comparison)
- [ ] Word count meets target for content type

## Meta Tags

- [ ] Title tag: primary keyword near front, under 60 chars
- [ ] Meta description: includes keyword, has CTA, under 155 chars
- [ ] OG title and description set

## Structure

- [ ] Single H1
- [ ] Logical heading hierarchy (no skipped levels)
- [ ] Short paragraphs (3-4 sentences max)
- [ ] Bullet lists or tables where appropriate

## Links & Media

- [ ] Internal links to 2-3 related articles
- [ ] Affiliate links use `rel="nofollow sponsored"`
- [ ] Images have descriptive alt text
- [ ] No broken links

## Final Check

- [ ] Reads naturally — no keyword stuffing
- [ ] Voice matches the content workspace voice guide
- [ ] Fact-checked: regions, producers, vintages are accurate
