## Gap analysis — what top pages have that we don't

1. **Comparison table at top of page (Cellar Beast):** 5-column quick-reference table (Varietal / Price Range / Tasting Notes / Why Beginners Love It) before the prose begins. We don't have any consolidated table. This is the single strongest structural feature on the SERP for this keyword.
2. **Body-weight grouping H2s (Good Pair Days, Cellar Beast):** Good Pair Days splits by Light / Medium / Full. We label wines as light/medium/full in the WineQuiz tags but don't group the body listings that way. A beginner learning "I prefer light reds" needs category H2s to jump to.
3. **Grape-first structure instead of bottle-first (Good Pair Days, Wine Ninjas, Cellar Beast, Lexi's Wine List):** 4 of 5 true-listicle competitors organise by grape variety first (1 H2 per grape), then suggest 1–2 bottles per grape. We organise bottle-first (1 H2 per bottle, with grape mentioned in the paragraph). For a "beginners" keyword, grape-first is arguably more pedagogical.
4. **Decanting / glassware / serving-temperature body sections (Cellar Beast):** Cellar Beast has dedicated subsections for Temperature Service, Glassware Selection, Decanting Decisions. We have serving temperature covered in a WineTip and that's it. For beginners specifically, these micro-topics matter.
5. **Sommelier / author-credentials E-E-A-T block (Cellar Beast has WSET Level III, SomeGoodCleanFun sober sommelier, Lexi's WSET):** Every serious competitor names a credentialed author. We have Claire Bennett as Wine Editor (generic). For beginner-intent queries where readers need trust, stronger credential signalling would help. (Not Lino's call — flag to Iris.)
6. **Per-wine price range displayed inline (Cellar Beast table, Lexi's):** Standard across the SERP for beginner pages. We use "Check Price" strategy — note but don't close.
7. **Explainer content about varietal character, regions, food pairing theory (all competitors):** We actually have excellent secondary-section coverage here. Not a gap — possibly our strongest zone.

**What nobody has that we already do:** WineQuiz, MethodologyBox, per-wine critic-score ring charts + 4.3–4.9-star buyer ratings visible, Top 3 sidebar, umbrella cross-links to sibling category pages.

**Add a "Beginner Gateway Map" — a visual progression chart** showing the reader a 3-tier recommended journey through our 8 bottles:

- **Tier 1: Start here (3 bottles)** — lowest tannin, most fruit-forward, most forgiving. Sur de los Andes Pinot, BenMarco Malbec, Escudo Rojo Pinot.
- **Tier 2: After your first 3 (3 bottles)** — introducing structure, earthiness, oak. Castellani Chianti Classico, Vinos de Arganza Mencia, Chateau Bourdieu Bordeaux.
- **Tier 3: Graduation bottles (2 bottles)** — stepping up body and tannin. J. Lohr Seven Oaks Cab, Frescobaldi Nipozzano Rufina.

Implementation: A 3-column CSS grid or 3-row table with "Stage / Bottles / What you're learning" columns. Frame it as a reading-order recommendation, not a ranking.

Wine Ninjas counts down 5→1 but the ranking is based on smoothness, not learning path. A "do this first, then this" gateway map is the single most useful thing you can give a beginner — and it directly differentiates from the grape-list + grape-table conventions saturating the SERP.

**Bonus novelty:** Add a comparison table matching Cellar Beast's format (Varietal / Price / Tasting Notes / Why Beginners Love It) but using our 8 specific bottles, not grape categories. We'd be the only page combining specific-bottle recommendations with a scannable table. All competitors do one or the other.
