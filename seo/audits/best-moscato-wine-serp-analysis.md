## 6. Novel feature recommendations (at least one competitor does not have)

All four are genuinely novel across the entire top 9 — **zero competitors have anything like these.**

### Novel #1 (recommended primary): **Sweetness × Fizz positioning matrix**

A `<DotScaleRow>`-style 2D matrix that plots each of our 12–15 wines on a single chart:
- **X-axis:** sweetness (1 = off-dry / 5 = dessert-sweet)
- **Y-axis:** fizz level (0 = still / 1 = frizzante / 2 = fully sparkling)
- Each wine is a labelled dot. Style clusters emerge visibly: Moscato d'Asti (frizzante, medium-sweet), Asti Spumante (fully sparkling, medium-sweet), Passito (still, ultra-sweet), Pink Moscato (fully sparkling, sweet), etc.

**Why this works for Moscato specifically:** the #1 reader-confusion axis in this category is "which one's sparkling?" and "how sweet is it really?" No competitor answers both visually on the same page. Maps directly to master-audit Pattern B ("2D positioning matrices absent from every competitor SERP").

### Novel #2: **"Moscato style map" — Moscato d'Asti vs Asti vs Pink vs Passito vs Still reference card**

A compact reference component above the wine list showing the five Moscato sub-styles with:
- Typical ABV range
- Fizz level
- Sweetness range
- Typical price band
- 1-line pairing note

Solves the Wine-Searcher "Moscato d'Asti vs Asti Spumante" gap AND the reader's "which one do I want?" problem in a single glance. No competitor has a multi-style comparator — wine-searcher has a prose explainer only.

### Novel #3: **ABV ladder / low-ABV badge**

Moscato's defining trait is its low alcohol (5–6.5% for Moscato d'Asti vs 11–14% for typical white). Nobody surfaces this visually.
- Add an "ABV %" field to the `<WineDataBlock>` micro-spec strip
- Add a "Low-ABV pick" badge (e.g. ≤6.5%) as a `highlight` prop
- Add a mini-section or FAQ answering "Moscato with the lowest alcohol" — since this is a top use-case (weddings, brunch, day-drinking)

Direct intent match for low-ABV seekers. Zero effort, distinct editorial angle.

### Novel #4: **Food pairing matrix (spicy / dessert / brunch / fruit / cheese)**

A 5×N matrix where rows are our wines and columns are pairing contexts (spicy food, dessert, brunch, fresh fruit, blue cheese). Checkmarks or 1-5 fit score per cell.

Expands Gambero Rosso's single-pairing-theme angle (Piedmontese dessert) into a multi-occasion reference chart. Moscato is notably flexible for pairing — this surfaces that flexibility in a scannable matrix. Zero competitors have a pairing matrix for Moscato.

**Recommended: ship Novel #1 (Sweetness × Fizz matrix) + Novel #2 (Moscato style map reference card) minimum. Novel #3 (Low-ABV badge) is a prop-level add, ship it too. Novel #4 is bonus if component budget allows.**
